# mark2doc

将markdown生成单html的文档项目

## 安装

```shell
go install github.com/yqchilde/mark2doc
```

## 用法

```shell
用法: mark2doc -i <markdown文件> -o <html文件> [-title <自定义标题>]
  -i string
        输入Markdown文件路径 (必填)
  -o string
        输出HTML文件路径 (必填)
  -title string
        自定义HTML文档标题 (可选，默认使用文件名)
```
